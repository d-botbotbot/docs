# Command Reference
Moderator commands can be found [here](moderator).

## No Category:
Uncategorized Commands

### `help`

Show a list of commands for the bot

Required Permission: None

#### Syntax

```
+help
```


#### Examples

```
+help
```

### `info`

Get info about the bot

Required Permission: None

#### Syntax

```html
+info
```

#### Examples

```
+info
```
